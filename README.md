Bem-vindo(a) ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido do zero para apresentar minhas habilidades como **Front-end Developer** e **UI Designer**, além de expor os projetos reais e conceituais que venho construindo.

## 🚀 Sobre o Projeto

O objetivo principal deste site é servir como minha vitrine profissional para captação de clientes freelancers e conexão no LinkedIn. Foquei em criar uma experiência visual **premium**, limpa e totalmente voltada para a conversão de novos negócios.

### 🎯 Diferenciais do Projeto
* **Identidade Visual Exclusiva**: Desenvolvida com foco em estética minimalista e moderna.
* **Seção de Destaque Interativa**: Apresentação de projetos principais através de um mockup de navegador funcional.
* **Performance Absoluta**: Código limpo, sem excesso de bibliotecas pesadas e carregamento rápido.
* **100% Responsivo**: Totalmente adaptado para rodar perfeitamente em celulares, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

Para garantir um código semântico, performático e de fácil manutenção, utilizei a seguinte stack técnica:

* **HTML5** — Estruturação semântica e acessível.
* **CSS3** — Grid, Flexbox, variáveis CSS e Media Queries avançadas (sem frameworks e sem o uso de `@import` para manter a performance).
* **JavaScript (Vanilla)** — Lógica de interações, manipulação de DOM e efeitos de rolagem.
* **Intersection Observer API** — Para otimização de animações baseadas no scroll do usuário.
